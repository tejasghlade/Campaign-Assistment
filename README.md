## Installation

Instructions on how to install and set up your project.

```bash
# Example
git clone https://github.com/tejasghlade/Campaign-Assistment/tree/main
cd Campaign-Assistment 
```

frontend
```bash
cd frontend
npm install
npm start
```

backend

- Add Environment Variables for postgresql database

```bash
cd backend
npm install
npm run start:dev
```
